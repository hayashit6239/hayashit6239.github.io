# Google Cloud Technical Summit 2025 powered by Jagu'é'r

技術カンファレンスのティザーサイト

## 技術スタック

- **React** - UIライブラリ
- **Next.js (App Router)** - フレームワーク
- **Mantine** - UIコンポーネントライブラリ
- **Tailwind CSS** - CSSフレームワーク
- **GitHub Pages** - ホスティング

## 開発

### セットアップ

```bash
npm install
```

### 開発サーバー起動

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

### ビルド

```bash
npm run build
```

## デプロイ

mainブランチへのプッシュで自動的にGitHub Pagesにデプロイされます。

## サイト構成

- ヒーローセクション
- 概要 (実施日時、対象者、参加方式)
- カンファレンステーマ
- CfP募集
- 行動規範
- スタッフ紹介
